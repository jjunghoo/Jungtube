# Jungtube

## 설명
YouTube API를 활용한 나만의 Youtube 사이트<br /><br />

## Demo 
https://jjunghoo.github.io/jungtube

<br />

## 사용기술

프로그래밍 언어 : JavaScript

UI 라이브러리 : React

스타일링 : PostCSS

상태 관리 : React Hooks

빌드 도구 : Create React App

배포 : Github Pages 활용

<br />

## 개선사항
- 댓글더보기에서 해당댓글에 대한 대댓글들만 불러와져야한다.
- 전체적인 UI가 좀 더 이쁘게 스타일링이 되어야 할 것 같다.
